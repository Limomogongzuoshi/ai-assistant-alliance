## 快速部署（5分钟完成）

### 步骤 1：上传文件

1. 打开 [github.com/Limomogongzuoshi/ai-assistant-alliance](https://github.com/Limomogongzuoshi/ai-assistant-alliance)
2. 点击 **"Add file"** → **"Upload files"**
3. 将以下文件拖拽上传：
   - `index.html`（主页面）
   - `PITCH_DECK.md`（PPT 内容）
4. 点击 **"Commit changes"**

### 步骤 2：启用 GitHub Pages

1. 进入仓库 **Settings**（设置）
2. 左侧菜单找到 **"Pages"**
3. 设置选项：
   - **Source**: 选择 **"Deploy from a branch"**
   - **Branch**: 选择 **"main"**，文件夹选 **"/ (root)"**
4. 点击 **"Save"**

### 步骤 3：获取链接

等待 1-2 分钟后，刷新页面，顶部会显示：

```
Your site is live at https://limomogongzuoshi.github.io/ai-assistant-alliance/
```
